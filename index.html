Desenvolvimento de um Painel Gerencial para Marketplace de Entregas com o Streamlit

Neste projeto, foram aplicados conceitos de Programação em Python, manipulação de dados, pensamento estratégico e lógica de negócios, aliados a ferramentas de desenvolvimento web como Streamlit e GitHub. O objetivo foi desenvolver um painel gerencial com as principais métricas de uma empresa de marketplace de delivery de comida.
